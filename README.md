# Ranking with MWFAS

Code and data for the paper **Ranking with Minimum Weight Feedback Arc Set**.

## Reference

- **Paper:** https://arxiv.org/html/2412.16181v1  
- **Original data:** https://github.com/SherylHYX/GNNRank (graphs converted to edge-list format and included in this repo)

## What is in this repo

- Code for ranking methods based on minimum weight feedback arc set (MWFAS).
- Data in edge-list format.
- Scripts to reproduce experiments from the paper.

## How to run

Clone the repo, install dependencies, run the main script with the desired dataset. See the paper and in-repo docs for hyperparameters and evaluation.
